## Dataset

El dataset utilizado en este proyecto es `036ei5wg8bzm6bfnggh2.csv`, que contiene datos de mortalidad en Chile entre los años 1997 y 2019.
El dataset incluye las siguientes columnas:
- `gru_edad`: Grupo etario
- `sexo`: Sexo del paciente
- `ano`: Año de registro
- `nombre_enfermedad`: Nombre de la enfermedad
- `n`: Número de casos

## Cómo Replicar

### 1. Clonar el Repositorio

```bash
git clone <URL_DEL_REPOSITORIO>
```

### 2. Instalar Dependencias

Asegúrate de tener Python 3.10 o superior instalado. Luego, instala las dependencias usando pip:

```bash
pip install -r requirements.txt
```

### 3. Ejecutar la Aplicación

La aplicación Shiny se encuentra en la carpeta `web_app`. Para ejecutarla, navega a esa carpeta y ejecuta el siguiente comando:

```bash
cd web_app
shiny run app.py
```

La aplicación estará disponible en `http://127.0.0.1:8000`.
